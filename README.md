# About
- This project is a calculator in rescript language

# Requirements
- NodeJS
- Rescript 

# How to run this projects

Install all dependecies
```sh
npm install
```
Run server in your browser on `http://localhost:3000/`
```sh
npm run server
```
In other terminal compile for JS
```sh
npm start
```
Or
```sh
bsb -make-word
```
